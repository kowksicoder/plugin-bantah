# Registry PR Body

## Title

`feat: add @elizaos/plugin-bantah to registry`

## Body

This PR adds `@elizaos/plugin-bantah` to the elizaOS plugin registry.

### Plugin

- NPM package: `@elizaos/plugin-bantah`
- Repository: `github:kowksicoder/plugin-bantah`

### What it does

This plugin gives Eliza agents Bantah prediction market actions:

- `create_market`
- `join_yes`
- `join_no`
- `read_market`
- `check_balance`

It is designed for agents that want to interact with Bantah-compatible runtime endpoints and participate in Bantah markets.
